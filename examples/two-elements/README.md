# Two elements example

Two small examples showing how the size of the input domain affects external
equivalence.

```bash
# Equivalent only for domains with at least two elements
anthem-cx two-elements.1.lp two-elements.2.lp two-elements.ug

# Equivalent only for domains with fewer than two elements
anthem-cx two-elements.1.lp two-elements.3.lp two-elements.ug
```

See the
[documentation](https://docs.potassco.org/anthem-cx/examples/two-elements/) for
a detailed explanation.

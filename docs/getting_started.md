# Getting started

## Installation

=== "Pip"

    ```console
    pip install anthem-cx
    ```

=== "Development mode"

    ```console
    git clone https://github.com/potassco/anthem-cx.git/
    cd anthem-cx
    pip install -e .[all]
    ```

    !!! warning
        Use only for development purposes

## Usage

### Command line interface

Details about the command line usage can be found with:

```console
anthem-cx -h
```

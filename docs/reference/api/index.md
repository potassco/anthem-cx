# API

The API reference is organized to mirror the package structure of
*anthem-cx*. This page documents the top-level `anthem_cx` package and its
high-level public API; the pages below cover the individual modules and
subpackages.

- **[cx_program](cx_program.md)** — core algorithm: normalization, public
  reduct, and CX program assembly.
- **[Analysis](analysis/index.md)** — conflict, dependency, and input
  analysis.
- **[Transformation](transformation/index.md)** — the AST transformer
  pipeline.
- **[Utils](utils/index.md)** — shared data types, solving, parsing, output,
  logging, and errors.
